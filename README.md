# E-commerce SQL Analytics Project

This repository contains a SQL-based data analysis project built on a
simplified e-commerce dataset.

The main goal of this project is to demonstrate analytical thinking,
SQL skills, and the ability to answer business questions using data.

## Tools Used
- PostgreSQL
- pgAdmin
- SQL

## Project Structure
- data/ - raw CSV files used in the analysis
- sql/ - SQL queries used to answer business questions
- ANALYSIS.md - step-by-step explanation of the analytical process

## Key Topics Covered
- Revenue analysis
- Average order value (AOV)
- Customer-level revenue
- Basic retention analysis
